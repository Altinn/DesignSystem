How to use the design system

Atomic design is a methodology used to construct web design systems. Read more about it here. You can find all the components in the top navigation bar, organized into Atoms, Molecules, Organisms, Templates and Pages.

Keyboard Shortcuts

ctrl+shift+a: open/close annotations view

The goal

The goals for us in creating this type of living Design System are:

  • Maintain a consistency of styles and identity in all Alltinn's web solutions
  • Having one common prototype environment where we are simultaneously able to see components abstractly and in context
  • Streamline the workflow for designers and developers
  • Create a focal point for our Design and User Experience work, both for ourselves and for the community
